#  Smart Parking System using ESP8266 and Blynk

An IoT-based Smart Parking System that detects vehicle occupancy using IR sensors and displays real-time parking status on the Blynk Cloud Dashboard.
---

##  Features

- Real-time parking monitoring
- Automatic vehicle detection
- Live Blynk dashboard
- Total occupied slot count
- Low-cost IoT solution
- WiFi-based monitoring

##  Hardware Used

- ESP8266 NodeMCU
- 4 × IR Sensors
- Breadboard
- Jumper Wires
- USB Cable

---

##  Software Used

- Arduino IDE
- Embedded C++
- Blynk IoT Platform

---

##  Working

1. IR sensors detect vehicles.
2. ESP8266 reads sensor values.
3. WiFi sends data to Blynk Cloud.
4. Dashboard updates parking status.
5. Total occupied slots are calculated automatically.
